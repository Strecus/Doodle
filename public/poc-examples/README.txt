Six demo POC upload files for DOODLE (500 positions each).

Wild = matches the in-app reference for that organism (no resistance SNPs in this window).
Resistant = 10 SNPs vs that same reference (demo only, not clinical isolates).

In the app: check "POC test completed", choose Gonorrhea / Syphilis / Chlamydia, then upload the matching wild or resistant file.

Only the gonorrhea upload changes the ceftriaxone vs dual-therapy demo line; syphilis/chlamydia results are informational on this screen.

Regenerate refs + files from repo root:  node Doodle/scripts/gen-poc-refs.mjs  (run inside Doodle folder).
